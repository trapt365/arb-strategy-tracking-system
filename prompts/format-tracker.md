## Инструкции

Сформатируй отчёт в **максимум 3 секции**:

### Секция 1: Ключевые решения и обязательства

Список решений и обязательств. Каждое обязательство в формате:
- **[Имя]** обязался: _что сделать_ | Срок: _дата_ | Цитата: _"..."_ [MM:SS]

Пометки:
- [approximate] — если цитата не дословная
- [speaker_check] — если спикер требует верификации

### Секция 2: OKR-покрытие

Краткая таблица:
- Обсуждённые KR (discussed) — какие именно, ключевые выводы
- Слепые зоны (blind_zone) — какие KR не затронуты

### Секция 3: Алерты и рекомендации

Только если есть существенные алерты из анализа.

## Формат ответа

Верни ТОЛЬКО валидный JSON (без markdown fences):

{
  "report_sections": [
    {
      "title": "Название секции",
      "content": "Отформатированный текст секции в markdown"
    }
  ],
  "summary_line": "Одна строка-резюме встречи (до 100 символов)",
  "commitment_count": 3,
  "alert_count": 1
}

## Критически важно

- Максимум 3 секции. Если алертов нет — 2 секции.
- Формат должен быть **сканируемым**: списки, bold, короткие предложения.
- Точность > Полнота. Не добавляй то, чего нет в данных.
- Язык отчёта — русский.
