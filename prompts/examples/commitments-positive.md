# Примеры правильно извлечённых обязательств

## Пример 1: Явное обязательство с дедлайном

**Транскрипт (фрагмент):**
> [12:45] Азиза: Я сегодня постараюсь дать вам обратную связь. До конца дня напишу.

**Извлечение:**
```json
{
  "who": "Азиза",
  "what": "Дать обратную связь по сообщению",
  "deadline": "сегодня, до конца дня",
  "quote": "[12:45] Я сегодня постараюсь дать вам обратную связь. До конца дня напишу."
}
```

## Пример 2: Обязательство с относительным сроком

**Транскрипт (фрагмент):**
> [25:30] Жумбак: Мен қазір мектеппен келісемін. Осы аптаның соңына дейін.

**Извлечение:**
```json
{
  "who": "Жумбак",
  "what": "Договориться со школой",
  "deadline": "до конца этой недели",
  "quote": "[25:30] Мен қазір мектеппен келісемін. Осы аптаның соңына дейін."
}
```

## Пример 3: Совместное обязательство после обсуждения

**Транскрипт (фрагмент):**
> [40:15] Азиза: Давайте тогда до 14-го числа поставим дедлайн.
> [40:20] Жумбак: Хорошо, до 14-го.

**Извлечение:**
```json
{
  "who": "Жумбак",
  "what": "Раскрыть ценность тренажёров для пользователей",
  "deadline": "до 14-го числа",
  "quote": "[40:15–40:20] Давайте тогда до 14-го числа поставим дедлайн. — Хорошо, до 14-го."
}
```

## Пример 4: Обязательство о делегировании

**Транскрипт (фрагмент):**
> [35:10] Жумбак: Я думаю сейчас передать работу именно договариваться, объяснять школам деген жұмысты көмекшіге өткізіп.

**Извлечение:**
```json
{
  "who": "Жумбак",
  "what": "Передать работу по школам помощнице",
  "deadline": "не указан",
  "quote": "[35:10] Я думаю сейчас передать работу именно договариваться, объяснять школам деген жұмысты көмекшіге өткізіп."
}
```
