# ARB Strategic AI-Tracking System

AI-платформа автоматизации контроля качества и коучинг-аналитики для стратегического трекинга.

## Проблема

Реализация стратегий проваливается не из-за плохих стратегий, а из-за недостатка организационной способности команд: планировать недельное исполнение, формулировать гипотезы и связывать задачи со стратегическими целями. Контроль качества трекинга выполняется вручную, что ограничивает масштаб до 2–3 клиентов.

## Решение

AI-пайплайн трансформирует транскрипты коучинговых сессий в actionable-аналитику для трёх стейкхолдеров:

- **Трекер (коуч)** — авто-отчёты, обратная связь по качеству, подготовка к встречам
- **Руководство ARB** — QC-дашборд по всем коучам
- **CEO-клиент** — видимость роста организационной способности команды

## Конкурентное преимущество

Полный цикл: стратегическая сессия → OKR → недельный коучинг → AI QC → рост организационной способности. Ни один конкурент (Gong, Viva Goals, BetterUp) не покрывает эту цепочку целиком.
