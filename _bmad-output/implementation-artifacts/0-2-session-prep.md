## Пояснение Leading vs Lagging (2.4)

> **Для Дамира — 1 абзац:**
>
> Мы будем собирать у каждого топа 2 метрики в неделю через Telegram-бот. **Leading** — это опережающий показатель, на который топ может влиять прямо сейчас (количество звонков, количество публикаций). **Lagging** — это результат, который видно с задержкой (выручка, NPS). Leading показывает «делаем ли мы правильные действия», lagging — «получаем ли правильные результаты». Нам нужно по одной каждого типа на департамент.

## Примеры F5 метрик из PRD (2.3)

| Департамент | Leading | Lagging |
|------------|---------|---------|

| Маркетинг | Кол-во лидов / CAC | Конверсия в покупку |
| HR | eNPS | Текучесть кадров |
| Финансы | Cash flow | Маржинальность |
| Продукт | DAU / MAU | NPS / Retention |
