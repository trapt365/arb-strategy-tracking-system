**Изменения vs session-prep:**
- **SuperApp:** было `DAU SuperApp` + `Конверсия в платные` → стало `DAU/MAU (Stickiness)` + `Revenue per DAU` (предложение Азизы)
- **Продажи:** было `Кол-во лидов` + `Выручка / ср. чек` → стало `Конверсия лид→продажа` + `Выручка + ср. чек` (предложение Азизы)
- **Маркетинг:** было `CAC` + `Кв. лидов` → стало `Кол-во лидов` + `CAC` (Азиза поменяла leading/lagging местами)

**Способ сбора (MVP):** Азиза спрашивает на еженедельных встречах и вносит в Google Sheets вручную. Метрики есть в РнП компании — подтверждено Азизой.

## 4. Google Sheets инфраструктура

**Apps Script:** `sheets/Code.js` — управляется через clasp, генерирует 3 скрытых листа:
- `_stakeholder_map` — заголовки: `full_name`, `speaker_name`, `department`, `role`, `bsc_category`, `responsibility_areas`, `interests`, `notes`
- `_okr` — заголовки: `kr_number`, `short_name`, `key_result`, `owner`, `owner_position`, `current_status`, `target`, `progress`, `deadline`, `okr_group`, `quarter`
- `_f5_metrics` — заголовки: `department`, `metric_name`, `metric_type`, `unit`, `source`, `owner_speaker_name`, `ranges`, `update_frequency`, `risk_notes`, `notes`

**Все заголовки snake_case** — контракт для Sheets adapter (Story 1.3).

**Sheet ID:** в `.env` (не коммитится). Placeholder в `.env.example`: `GEONLINE_F0_SHEET_ID=`

**Запуск:** меню «F0 Context» → «Обновить F0» в Google Sheets.
