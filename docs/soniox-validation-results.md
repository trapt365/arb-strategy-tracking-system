### Критические критерии

| Критерий | Порог | Результат | Статус |
|----------|-------|-----------|--------|
| WER на worst-case | < 15% | Ручная проверка владельцем: качество приемлемо | ✅ |
| Code-switching рус↔каз | Корректно на ≥3 записях | Обнаружен на 5 из 7 файлов (1100318212, 1111482399, 1554018312, 1663213769, 1951904349). Маркировка `language: kk/ru` присутствует на уровне токенов. Корректность пословной классификации требует выборочной проверки. | 🟡 (предв. ✅) |

### Важные критерии

| Критерий | Порог | Результат | Статус |
|----------|-------|-----------|--------|
| Полный поток (запись → JSON) | < 5 мин для 30 мин | ✅ С огромным запасом: 33 мин → 72с, 41 мин → 79с, 50 мин → 105с (~25–30× быстрее реального времени). Только транскрипция, без скачивания из Google Drive. | ✅ |
| Формат Google Meet совместим | webm/mp4 принимается | m4a: 7/7 OK. mp4 218MB — ошибка формата, не критично для MVP (m4a = основной формат) | ✅ (m4a) |
| Webhook работает | completed + error | Отложен — не блокер для Go (polling работает) | ⬜ → backlog |
| Fallback friction | < 2 мин | Отложен — не блокер для Go | ⬜ → backlog |

- **audio1554018312.m4a:** баланс спикеров `{1: 4000, 2: 6, 3: 9120}` — Speaker 2 имеет всего 6 токенов, вероятна ложная сегментация (фактически 2 спикера, не 3).
- **audio1663213769.m4a:** баланс `{1: 1801, 2: 3896, 3: 8, 4: 233}` — Speakers 3 и 4 похожи на over-segmentation; реально, вероятно, 2 спикера.
- **audio1602529797.m4a / audio1721976611.m4a:** определены как **только казахский** (kk без ru). Нужна проверка: действительно ли нет русских вставок, или language detection пропускает их.

## Формат API ответа (Soniox)

```json
{
  "id": "uuid",
  "text": "полный текст",
  "tokens": [
    {
      "text": "слово",
      "start_ms": 0,
      "end_ms": 450,
      "confidence": 0.97,
      "speaker": "1",
      "language": "ru",
      "is_audio_event": false
    }
  ]
}
```

## Маппинг на Transcript Interface Contract

| Soniox | Interface Contract | Преобразование |
|--------|-------------------|----------------|
| tokens[].speaker | speakers[].name | Группировка по speaker, name = "Speaker {N}" |
| tokens[].start_ms, end_ms | segments[].start, end | Объединение последовательных tokens одного speaker |
| tokens[].text | segments[].text | Конкатенация текстов в сегменте |
| max(end_ms) | metadata.duration | Максимальное end_ms |
| tokens[].language | — | Доп. метаинформация (не в контракте) |
