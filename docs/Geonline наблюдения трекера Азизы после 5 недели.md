# Наблюдения после 5-й недели трекинга

## 1. Командная динамика и культура

В команде сохраняется низкий уровень доверия между сотрудниками.

По словам участников команды, одной из причин является сомнение в компетенциях некоторых коллег, из-за чего сотрудники:

- предпочитают выполнять задачи самостоятельно  
      
    
- редко обращаются за помощью  
      
    
- не проводят совместные обсуждения и брейнштормы  
      
    

Это снижает скорость решения задач и ограничивает появление новых идей.

Гипотеза HR:  
Проводить регулярные встречи в узком кругу топ-менеджмента, где команда будет:

- обсуждать идеи  
      
    
- проводить брейнштормы  
      
    
- совместно искать решения ключевых проблем  
      
    

Цель — создать безопасную среду для открытого диалога и усилить командное взаимодействие.

# 2. Управление и операционные процессы

### Еженедельная отчетность по метрикам

У команды есть потребность в регулярной синхронизации по ключевым метрикам департаментов.

Предлагается внедрить еженедельный короткий отчет (5-минутка) по основным показателям, пример:

Отдел продаж

- количество продаж  
      
    
- источники продаж  
      
    
- конверсия  
      
    

Маркетинг

- активные креативы  
      
    
- количество лидов  
      
    
- рекламный бюджет  
      
    

Рекомендуется разработать единый шаблон отчетности по департаментам.

# 4. Маркетинг и PR

### Медленная работа с кейсами

Отдел маркетинга неоперативно публикует результаты студентов.

Проблема:

- один студент может обучаться в нескольких центрах  
      
    
- другие центры публикуют результат быстрее  
      
    

В итоге:

- их кейс воспринимается как результат конкурентов
